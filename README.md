# 🎬 Movie App

A modern Full Stack movie web application that allows users to explore, search and discover movies through a dynamic and responsive interface.

## 🚀 Features

* Movie search functionality
* Dynamic API integration
* Responsive design
* Real-time content rendering
* Modern UI/UX
* Fast and optimized frontend
* User-friendly navigation

## 🛠️ Technologies Used

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3
* Tailwind CSS
* Bootstrap
* Redux

### Backend

* Node.js
* Express.js
* REST APIs

### Database

* PostgreSQL

### Tools

* Git
* GitHub
* Postman
* Vite
* VS Code

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Tomim112/movie-app.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
